# EvSync

An Event Collaboration app for to make your day easier

### Key takes
- Create Projects/Events
- Bring your friends
- Assign tasks
- Never forget things after commiting
- Track progress & backlog

### Tech stack
- Android Ecosystem (kotlin)
- Firebase
- Google calender API
